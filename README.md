## yum.sh

- 脚本说明: 一键更新yum源以及安装常用包
- 使用./yum 0安装基本包，使用./yum 1安装完整包

#### 下载安装:
``` bash
wget -N --no-check-certificate https://raw.githubusercontent.com/qxzg/shell/master/yum.sh && chmod +x yum.sh
```
